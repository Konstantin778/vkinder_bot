### Чат-бот Vkinder подбирает пару пользователю на основании информации его анкеты.

*	Бот с помощью VkApi считывает информацию о пользователе при запуске бота

*	На основании полученной информации о городе проживаний, возрасте, семейном положении и поле бот осуществляет поиск пользователей, записывая данные в список.

*	При каждом нажатии кнопки «показ» бот выводит имя, ссылку на страницу и три самые популярные фотографии одного найденного человека из списка.

*	Данные об id пользователя, а также об имени и id найденного человека записываются в базу данных для каждого пользователя бота.

*	При выводе результатов бота и записи их в базу данных пользователи не повторяются.


Для старта бота и поиска пары в строке сообщений нужно написать "старт"
